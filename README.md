This is an ongoing effort to implement Prism in Rust.

# Build

Run `cargo build`.

# Run Tests

Run `cargo test`. To see all the outputs, append `-- --no_capture`.

# Visualization

Run `cargo run --example self_mining` and go to `localhost:8888`.
